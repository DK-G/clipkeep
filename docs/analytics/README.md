# ClipKeep GA4 Local Exports

This directory is for local Google Analytics 4 exports generated by:

```powershell
npm run analytics:ga4
```

Generated `.json` and `.csv` files are ignored by Git because they can contain business analytics data.

Required local environment file:

```text
.env.analytics.local
```

Required variables:

```env
GA4_PROPERTY_ID=528376605
GOOGLE_APPLICATION_CREDENTIALS=C:\dev\portfolio\web\clipkeep\.secrets\ga4-service-account.json
```
