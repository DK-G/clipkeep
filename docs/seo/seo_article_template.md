## 1) Quick Answer
Answer the query directly in 2-4 sentences.
State what works, what does not, and the fastest safe method.

## 2) Step-by-Step
1. Open source post/app.
2. Copy URL.
3. Open ClipKeep tool page.
4. Paste URL and run extraction.
5. Save result.

Add screenshots where possible.

## 3) Safety / Legal Notes
- Respect platform terms and copyright.
- Personal archiving vs redistribution distinction.
- Mention privacy-safe behavior.

## 4) Common Failures and Fixes
- Invalid URL
- Private content limitation
- App/browser cache issue
- Region/network issue
- Platform-side temporary restriction

## 5) CTA Block (Mandatory)
- Primary CTA button/link to one tool page.
- Supporting link to troubleshooting page.

## 6) Related Reads (Mandatory)
- 3+ internal links.
- Mix: one same-platform, one troubleshooting, one comparison.

## Writing Constraints
- Keep paragraphs short.
- Use exact keyword in H1 and first 100 words.
- Use related variants naturally in H2/H3.
- Avoid overpromising unsupported behavior.

## QA Checklist Before Publish
- Keyword in title, intro, one H2.
- Meta title and description drafted.
- At least 2 internal links + 1 CTA to tool page.
- No factual conflict with current product behavior.
- Locale-ready wording (avoid slang that breaks translation).
