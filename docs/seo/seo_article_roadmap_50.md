## Internal Linking Rules (Mandatory)
- Every post must include a primary CTA to one tool page.
- Every post must include at least 2 internal links:
  - One to a tool page (`/download-twitter-video`, `/download-tiktok-video`, `/download-telegram-video`)
  - One to a related troubleshooting / guide page (`/solution/*` or related blog post)
- Add a related-post block at bottom (3 links minimum).

## Post Template Rules (Mandatory)
Each post must include these sections in order:
1. Quick answer (1-2 paragraphs)

3. Safety / legal notes
