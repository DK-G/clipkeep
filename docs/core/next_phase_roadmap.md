# ClipKeep 次フェーズ改善ロードマップ（統合版）

## 目的
- 既存基盤を活かし、PV増加と回遊率向上を主目的に完成度を引き上げる。

## Phase 1: UX強化（CTR / 回遊率）
1. Heroに対応SNS表示（Twitter / TikTok / Telegram）
2. カード全体クリック対応（CTAは維持）
3. カード情報強化（platform badge / duration / download count）
4. スマホUI最適化（1列カード、入力幅100%、大きいCTA、広告サイズ）

## Phase 2: SEO強化
5. SEO本文に見出し構造を導入（H2/H3）
6. Trending -> Downloader の内部リンクを追加
7. `generateMetadata` によるページ別title/description最適化

## Phase 3: PV増加構造
8. Trending / Latest 導線を継続強化
9. 時間フィルター導入（Today / Week / Month）
10. Similar Videos 導入（ダウンロード後・詳細下）

## Phase 4: 信頼性
11. Aboutページ強化（概要/対応SNS/使い方/Privacy stance）
12. Contactページ強化（support/feedback/DMCA）

## Phase 5: 広告最適化（最終段階）
- Hero下 / カード中段 / 記事末の段階配置
- モバイルは 320x100 / 300x250 を中心に実測調整

## 完成形サイト構造
- Home
- Downloader
- Trending
- Latest
- About
- Contact

## KPI
- CTR（Tool導線）
- Pages / Session
- Organic流入
- 主要ページのCVR（extract submit / completed）
