# GA4 Weekly Review Template (ClipKeep)

最終更新: 2026-03-15

## 対象期間
- Week: YYYY-MM-DD 〜 YYYY-MM-DD

## 入力値（GA4）
- `gallery_card_click`:
- `download_click`:
- `similar_click`:

## 派生指標
- Gallery→Download率 = `download_click / gallery_card_click`
- Download→Similar率 = `similar_click / download_click`
- 先週比（各イベント件数）

## セグメント確認
- platform別（twitter / telegram / tiktok）
- locale別（en / ar）
- gallery_section別（recent / trending）

## 判定
- 良化:
  - Gallery→Download率 が先週比 +10%以上
  - Download→Similar率 が先週比 +10%以上
- 悪化:
  - いずれかが先週比 -10%以下

## アクション
- 良化時:
  - 伸びた導線を他ページへ横展開
- 悪化時:
  - クリック箇所の文言/配置見直しを1件起票
  - 翌週にA/B候補を1つ投入
