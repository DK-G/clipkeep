## 調査で判明した事実（設計の出発点）
- `src/lib/auto-trend.ts` `runAutoTrendUpdate()` は X トレンド**キーワード**（twittrend.jp の `.trend a`）を取得しているが、
  抽出ジョブ生成（`createJob(platform, url, "ja")`）後に**キーワードを破棄**している。TikTok 側はトピック名すら拾っていない。
  → 「{トレンド} video download」ページにはトピック名が必須。**まずトピックを永続化する**のが前提条件（設計の Phase 0 の核）。
- 動画単位ページ `/result/[jobId]` は sitemap 非収録・実質孤立（揮発的なので妥当）。これを鮮度 SEO の単位にすると
  薄ページ量産＝ガードレール #1（ドアウェイ/自動生成スパム）抵触。
- 律速はインデックス反映遅延（週次 #002）。薄ページ量産はクロールバジェットを食い律速を**悪化**させる純リスク。
