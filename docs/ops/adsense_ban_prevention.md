# AdSense BAN対策 運用ガイド（ClipKeep）

## 目的
- AdSense審査/運用での停止・BANリスクを最小化する。

## 禁止・注意事項
- 著作権侵害を助長する表現を避ける。
- 誤解を招くCTA（広告を機能ボタンに見せる）を禁止。
- 過剰広告（コンテンツより広告が目立つ配置）を禁止。
- 自己クリック誘導、報酬誘導、クリック乞食文言を禁止。

## ページ品質基準
- 法務ページを常時公開（Terms/Privacy/Cookies/DMCA）。
- 問い合わせ導線を明示。
- 内容の薄いページ量産を避ける。
- 独自価値（ガイド・FAQ・トラブル対応）を維持。

## 広告実装ルール
- Above the fold は1枠まで。
- メインCTA近傍に誤クリック誘発配置をしない。
- モバイルでの全画面占有を避ける。
- CLS悪化を防ぐため広告領域サイズを予約。

## 監視項目（週次）
- AdSenseポリシー警告有無
- 無効トラフィック兆候
- 直帰率/滞在時間の急変
- ページ品質（低品質ページの増加）

## インシデント対応
1. 警告発生ページの広告を一時停止。
2. 問題表現・配置を修正。
3. 修正記録を残し再審査申請。
4. 再発防止ルールを本書へ追記。
